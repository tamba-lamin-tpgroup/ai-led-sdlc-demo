# FambulTik brand assets

Place the following files here (sourced from design.tpgroupsl.com/foundations/logo-usage):

- `fambultik-wordmark-dark.svg` — dark variant (used in NavBar on light backgrounds)
- `fambultik-wordmark-light.svg` — light variant (used in hero on dark backgrounds)
- `tpgroup-wordmark.svg` — TpGroup parent wordmark (used in footer)

Minimum display sizes: 24 px tall on mobile, 32 px on desktop.
Never stretch, recolour, rotate, or place over busy imagery without an overlay.
